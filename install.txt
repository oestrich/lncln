REQUIREMENTS
------------

lncln needs PHP, MySQL, and ImageMagick.  So far only PHP5 and MySQL 5 have been tested.


INSTALLATION
------------

Upload all contents to your webserver.
Make sure that the images/ folder and all subfolders are writable by the webserver.
Rename config-sample.php to config.php and edit accordingly. Database settings, server, and URL need to be edited.
Query the database with the contents of mysql.sql

Go ahead and login!  Your default user is 'admin' with a password of 'password' (no quotes).
